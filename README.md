## To emphasize text
* *single asterisks* - 기울임체
* _single underscores_ - 기울임체
* **double asterisks** - 굵은글씨체
* __double underscores__ - 기울임체/굵은글씨체
* ***triple underscores*** - 기울임체/굵은글씨체
* ~~cancelline~~ - 취소줄        

        *single asterisks* - 기울임체
        _single underscores_ - 기울임체
        **double asterisks** - 굵은글씨체
        __double underscores__ - 기울임체/굵은글씨체
        ***triple underscores*** - 기울임체/굵은글씨체
        ~~cancelline~~ - 취소줄
